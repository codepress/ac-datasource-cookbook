# Recipes

The recipes are ordered from simplest to most advanced. If you are new to
Data Sources, start at the top and work your way down — each recipe builds
on the concepts introduced by the previous one.

| # | Recipe | What it teaches |
| --- | --- | --- |
| 01 | [Super Simple](01-super-simple.md) | The absolute minimum: register a table as a Data Source with no configuration. |
| 02 | [Human-Readable Labels](02-human-readable.md) | Human-readable column labels plus a couple of typed columns. |
| 03 | [Defined Columns](03-defined-columns.md) | Declare columns explicitly and filter unwanted columns out of the table. |
| 04 | [Table Relation](04-table-relation.md) | Join a second table into a Data Source via a `has_one` relation. |

Each recipe page contains:

- A short explanation of what the recipe demonstrates and the key takeaway.
- The full PHP source of the recipe, rendered inline.
- Links to the relevant entries in the [API reference](../api/index.md).
